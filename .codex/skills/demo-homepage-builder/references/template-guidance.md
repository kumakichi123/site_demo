# テンプレガイダンス

## 目的

営業先の事業者が見たときに、すぐ次を感じるリメイクを作る。

- `今のサイトよりかなり良い`
- `ちゃんと自分の業種に合っている`
- `自社情報に差し替えればそのまま使えそう`

## 入力の分け方

入力は次の3種類に分ける。

- 営業先の事実情報
- 参考サイト由来の判断
- リメイク用の訴求文

### 営業先の事実情報

- `{{company_name}}`
- `{{region}}`
- `{{phone}}`
- `{{email}}`
- `{{address}}`
- `{{primary_service}}`

### 参考サイト由来の判断

- `{{reference_url}}`
- `{{reference_header_style}}`
- `{{reference_hero_structure}}`
- `{{reference_cta_pattern}}`
- `{{reference_section_order}}`

### リメイク用の訴求文

- `{{hero_catch}}`
- `{{subcatch}}`
- `{{cta_label}}`
- `{{trust_point_1}}`
- `{{trust_point_2}}`

## 基本構成

共通骨格は次を基本にする。

1. ヘッダー
2. ヒーロー
3. 信頼要素
4. サービスまたは事例
5. 会社概要
6. CTA

その上で、業種に必要なセクションだけを追加する。

## 品質基準

- 1画面目の情報階層が明確
- スマホでも使える
- 文字切れや重なりがない
- 捏造表現がない
- 参考サイト由来のブランド要素が残っていない
- あとで部品化しやすいセクション境界になっている
