# デモホームページビルダー

本番サイトではなく、営業用の `リメイクテンプレ` を作る。

## 手順

1. `references/template-guidance.md` を読む
2. 営業先サイトと参考サイトを分けて観察する
3. 強い参考サイトを取り込み、テンプレ初稿にする
4. 営業先の事実情報と安全な文面へ差し替える
5. セクション境界を再利用しやすく保ったまま整える
6. PC とスマホで確認する

## ルール

- 会社情報を捏造しない
- 他社のロゴ、社名、実績、証言、連絡先を残さない
- セクションは監査、並び替え、再利用がしやすい形にする
- 薄い下層ページを増やすより、強いトップページを優先する
- 既存サイトより良くなったことが 1 画面目で伝わるようにする
